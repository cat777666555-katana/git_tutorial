# Markdown Preview Enhanced

## Installing

1.vscode拡張機能のInstalling Markdown Preview Enhancedをinstall
　※v0.8.18以降にすること。できれば最新に。脆弱性対応されているので
2.https://plantuml.com/downloadから、plantuml-mit-1.2026.3.jar をDL
　※MITが単純にライセンス的に厳しくないため選択している
3.vscodeのGUIで設定(Ctrl+,)
　VSCode左下の歯車アイコンから「設定」を開きます。
　検索バーに「markdown-preview-enhanced.plantumlJarPath」と入力。
　表示された項目に先ほど控えたplantuml.jarの絶対パスを入力
